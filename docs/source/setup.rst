Setup
-----

Dependencies
~~~~~~~~~~~~

The following packages are required:

-  ``numpy``

-  ``tqdm``

-  ``pyvista``

-  ``nest_asyncio``

-  ``trame``

-  ``trame_vuetify``

-  ``trame_vtk``

Installation
~~~~~~~~~~~~

GeViewer can be installed using ``pip`` as follows:

.. code:: bash

   pip install geviewer

To uninstall:

.. code:: bash

   pip uninstall geviewer
